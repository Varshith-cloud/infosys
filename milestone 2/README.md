# Milestone 2 – Advanced Authentication System

## 📌 Description
This project extends the authentication system with advanced security features and readability analysis.

## 🚀 Features
- User Registration & Login
- OTP-based Email Verification
- Account Lock after 3 failed attempts
- Forgot Password with OTP
- Password Reuse Prevention
- Admin Dashboard
- Readability Analyzer (Text + File Upload)

## 🛠 Technologies Used
- Python
- Streamlit
- SQLite
- JWT
- Bcrypt
- Ngrok
- SMTP (Gmail)
- Plotly
